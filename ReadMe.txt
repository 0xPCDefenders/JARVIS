Jarvis is an AI assistant using gpt-3 that you can speak to in a conversational manner.

- Create an express server that connects to the OpenAI API. This server returns a response based on
the content of the request
- Create the client that sends a request and question to the OpenAPI server
- Change the client request interface to support voice integration (speech to text)
- Change the response interface to support voice integration (text to speech)
